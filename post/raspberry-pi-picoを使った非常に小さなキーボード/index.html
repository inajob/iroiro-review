Raspberry Pi Picoを使った非常に小さなキーボード
Raspberry Pi Pico 自作キーボード
Published: 2023-08-31

Raspberry Pi Pico Gets A Tiny Keyboard On Its Backから発見。画像もここから転載。

Raspberry Pi Picoを使ったとても小さなキーボード。小さいけど59キーあります。 スイッチのはんだ付けは大変そうですが、それ以外はRaspberry Pi Picoをつけるだけなので比較的簡単に組み立てできそうです。

プログラムはCircuitPythonで作られており簡単にカスタマイズできそうです。

しかしこれは、相当小さいですね・・
