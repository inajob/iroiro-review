QAZ配列のかわいらしいキーボード
自作キーボード QAZ
Published: 2023-11-27

GitHub - monokuroumu/Kazik: 10U keyboard with regular staggerから発見。画像もここから転載。

非常にコンパクトな自作キーボードの作例です。

小型の自作キーボードではよく採用されるQAZ配列を採用しているのも印象的です。 写真をよく見ればわかるのですがキーボード左側のA、Zキーが存在せず、これはAlt、Shift、と共用になっています。これらのキーはなにかのキーと組み合わせてのみ利用されるので、単純に押して離されたときのみ、AやZとして認識することで使い分けることが出来るのです。

コントローラはXIAO RP2040を使っています。
