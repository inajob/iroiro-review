Raspberry Pi Picoを搭載したZX Spectrumエミュレータ
Raspberry Pi Pico ZX Spectrum
Published: 2023-07-04

A Pico-Based ZX Spectrum Emulatorから発見。画像もここから転載。

Raspberry Pi Picoを搭載した画面付きのZX Spectrumエミュレータです。 コンパクトな筐体に画面もキーボードも搭載されており、可愛いです。

Raspberry Pi Picoは頑張るとHDMI出力ができるので、このようにグラフィックカード的な使い方もできて面白いです。
