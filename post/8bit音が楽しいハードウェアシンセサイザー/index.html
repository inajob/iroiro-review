Published: 2021-03-02

LIVEN 8bit warps - 8bit wave memory synthesizer/8ビット波形メモリシンセから発見。画像もここから転載。

見た目からしてかっこいい シンセサイザーです。 ELZ_1というシンセサイザーを作っているチームの新作です。
