USB Type-C端子に直接挿せる開発ボード
ATSAMD11C14 開発ボード ATSAMD11
Published: 2021-08-30

USB-C DiXi - arduino ARM SAM D11 USB stickから発見。画像もここから転載。

ATSAMD11を搭載したシンプルな開発ボードです。基板上にUSB Type-Cのパターンが形成されており、基板をそのままUSB Typc-Cポートに刺すことが出来る設計が面白いです。 プログラムメモリは16KB、SRAMは4KBと性能も十分です。

端子はI2Cとシリアル、電源とリセットしか出ていないので注意が必要です。
