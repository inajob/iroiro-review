5wayスイッチをたくさん使ったキーボード
自作キーボード
Published: 2024-10-18

GitHub - dlip/harite: Harite 5-way Switch Keyboardから発見。画像もここから転載。

5wayスイッチをたくさん使ったキーボードです。 シンプルな見た目ですが、左右にそれぞれ4つのジョイスティックとトラックパッドを搭載し、様々な操作が実現できます。

例では英字の入力をしていますが、日本語のフリック入力とも相性がよさそうに見えます。
