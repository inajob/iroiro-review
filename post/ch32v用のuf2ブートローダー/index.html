CH32V用のuf2ブートローダー
CH32V uf2
Published: 2024-05-23

GitHub - ArcaneNibble/wch-uf2: CH32V UF2 bootloaderから発見。画像もここから転載。

デバイスをUSBでパソコンに繋げると、マスストレージとして認識され、そこにファームウェアのファイルをコピーすることで書き込みを実現するuf2という仕組みがあります。

有名なところだとRaspberry Pi Picoなどに採用されていますが、それ以外にも様々な開発ボードがこの仕組みに対応してます。 ここで紹介するのはCH32V用のuf2ブートローダーです。

uf2サポートがあれば専用の書き込み機や書き込みソフトウェアなしにファームウェアを書き換えられるので便利です。
